## iOS 빌드 및 실행

### 릴리즈 모드로 실행
디버그 모드가 아닌 **릴리즈 모드**로 빌드해야 합니다.

### 무한 대기 문제 해결
`flutter run --release` 실행 시 "Installing and launching..." 단계에서 무한 대기하는 경우:

```bash
# 1. 연결된 기기 확인
cd flutter_app
flutter devices

# 2. stuck 프로세스 종료
pkill -9 -f "flutter.*00008110"
pkill -9 -f idevicesyslog

# 3. 클린 빌드
flutter clean
flutter pub get

# 4. 릴리즈 모드로 실행 (device-id는 flutter devices에서 확인)
flutter run --release -d [device-id]
```

### Xcode로 실행하는 경우
1. `flutter_app/ios/Runner.xcworkspace` 열기
2. 상단에서 실제 기기 선택
3. Product > Scheme > Edit Scheme > Run > Build Configuration을 **Release**로 변경
4. ⌘R로 실행
